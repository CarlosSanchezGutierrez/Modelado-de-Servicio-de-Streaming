# Modelado-de-Servicio-de-Streaming
Proyecto de programación orientada a objetos para modelar un servicio de streaming, gestionando películas y series. Cada video tiene un ID, nombre, duración, género, y calificación. Las series incluyen episodios con títulos y temporada. Implementa herencia y polimorfismo para la visualización y calificación de videos.

Este proyecto de Programación Orientada a Objetos (POO) implementa un modelo de servicio de streaming, abarcando películas y series. Utilizando conceptos fundamentales de POO como herencia, polimorfismo, y encapsulamiento, el sistema gestiona videos con atributos comunes como ID, nombre, duración y género. Las series tienen la característica adicional de episodios, cada uno con un título y temporada, lo que permite diferenciar entre los tipos de contenido y aprovechar la reutilización de código mediante herencia.

En cuanto a la funcionalidad, el sistema calcula la calificación promedio de cada video, proporcionando una escala de calificación del 1 al 5. Se implementan métodos para mostrar los videos generales y sus calificaciones, y específicamente los episodios de series. Además, se incluyen funciones para filtrar y mostrar solo películas con sus calificaciones.

Las clases fueron diseñadas para fomentar la extensibilidad y la flexibilidad, permitiendo que se puedan añadir nuevos tipos de videos o funcionalidades en el futuro sin grandes modificaciones. La correcta implementación de polimorfismo asegura que los métodos compartidos entre clases, como los de mostrar la información y calificación, funcionen de manera coherente tanto para películas como para series.

Este proyecto no solo refleja una implementación sólida de conceptos clave de POO, sino que también simula una plataforma de streaming real, demostrando habilidades en la estructuración de código limpio, reutilizable y mantenible. Ideal para mostrar competencias en desarrollo de software orientado a objetos y diseño de sistemas modulares.
